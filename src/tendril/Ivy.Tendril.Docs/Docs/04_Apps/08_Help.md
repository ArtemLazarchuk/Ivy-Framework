## Shortcuts

- `Ctrl+D` — Dashboard
- `Ctrl+P` — Drafts / plans
- `Ctrl+J` — Jobs
- `Ctrl+R` — Review
- `Ctrl+M` — **MakePlan** overlay

Full bindings appear in Help at runtime.
